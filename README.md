# 💻 스터디 이름
알고리즘 공부 & 코딩/쿼리 테스트 준비 스터디

</br>

## 📌 규칙
- 스터디 전날 밤 12시까지 사전에 정한 문제 1개를 풀고 깃허브에 백업합니다.
- 스터디 당일 디스코드에 모여 본인이 작성한 코드/쿼리를 해설합니다.

</br>

## 🧭 시간 및 장소
- 매주 M요일 O시: 코딩 테스트 1문제
- 매주 N요일 O시: 쿼리 테스트 1문제
- @Discord
